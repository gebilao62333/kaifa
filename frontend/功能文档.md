# 多客陪玩 - 前端功能文档

## 1. 项目概述

多客陪玩是一个陪玩服务平台的移动端 Web 应用，采用 Vue 3 + Vite 构建，提供用户注册登录、陪玩师搜索推荐、在线/线下陪玩预约、游戏派单、即时通讯、语音/视频通话、礼物打赏、会员体系等功能。

### 技术栈

- **框架**: Vue 3 (Composition API + `<script setup>`)
- **路由**: Vue Router 4 (Web History 模式)
- **状态管理**: Pinia
- **HTTP 请求**: Fetch API + Axios
- **实时通讯**: Socket.IO
- **构建工具**: Vite
- **样式**: Scoped CSS

## 3. 核心功能模块

### 3.1 用户系统

- **登录注册**: 手机号验证码登录（`/api/user/send-sms`、`/api/user/login-mobile`）
- **第三方登录**: `/api/user/login-third`
- **用户信息**: 获取用户信息（`/api/user/get`）、更新资料（`/api/user/update`）
- **关注体系**: 关注/取消关注、粉丝/关注列表
- **用户等级**: Lv.1-Lv.∞ 等级体系

### 3.2 首页

- **轮播Banner**: 滑动切换展示推广内容
- **快速导航**: 4个快捷入口（线下陪玩、线上陪玩、派单大厅、需求发布）
- **推荐陪玩师**: 卡片式展示推荐用户，包含头像、标签、服务类型、价格、在线状态

### 3.3 广场动态

- **动态Feed流**: 用户发布图文动态，支持图片展示
- **互动功能**: 点赞、评论、关注
- **预约按钮**: 可直接从动态跳转预约
- **图片预览**: 点击图片可预览

### 3.4 消息/即时通讯

- **会话列表**: 显示最近聊天会话
- **私聊**: 一对一文字聊天
- **消息类型**: 文字、图片、语音
- **消息状态**: 发送中、已发送、已读
- **撤回消息**: 支持消息撤回
- **未读计数**: 底部导航角标显示未读消息数

### 3.5 陪玩服务

- **申请陪玩师**: 选择线上/线下服务类型
- **游戏选择**: 支持王者荣耀、和平精英、英雄联盟等多款热门游戏
- **服务管理**: 陪玩师端查看和管理自己的服务
- **上下线**: 陪玩师可切换在线/离线状态

### 3.6 需求发布

- **服务类型**: 线上陪玩 / 线下陪玩
- **游戏选择**: 从预设游戏列表中选择
- **日期选择**: 月历视图，支持年月切换
- **时间选择**: 滚轮式时间选择器（开始时间 ~ 结束时间）
- **服务时长**: 自定义时长和预算
- **备注信息**: 选填，最多200字
- **预算显示**: 实时计算并显示预算金额

### 3.7 预约系统

- **创建预约**: 选择陪玩师、日期时间
- **预约管理**: 查看我的预约列表
- **预约状态**: 待确认、已确认、已取消
- **陪玩师操作**: 确认/拒绝预约

### 3.8 派单大厅

- **订单池**: 展示可接单的游戏订单
- **抢单**: 陪玩师可抢单
- **订单状态**: 待接单、进行中、已完成

### 3.9 钱包与支付

- **钱包概览**: 显示金币余额
- **充值**: 选择充值套餐，微信支付
- **提现**: 提现到微信/支付宝
- **交易记录**: 收入记录、支出记录、提现记录

### 3.10 礼物与红包

- **礼物列表**: 获取可用礼物
- **赠送礼物**: 在聊天/直播中赠送礼物
- **礼物背包**: 查看已收到的礼物
- **红包系统**: 发送/领取红包

### 3.11 VIP会员

- **会员套餐**: 多档VIP套餐
- **会员特权**: 专属标识、功能解锁
- **购买/续费**: 创建会员订单并支付

### 3.12 音视频通话

- **语音通话**: 一对一语音通话（基于TRTC）
- **视频通话**: 一对一视频通话
- **通话记录**: 通话历史

### 3.13 个人中心

- **个人信息**: 头像、昵称、等级、VIP标识
- **统计数据**: 关注、粉丝、获赞、访客
- **功能菜单**: 我的余额、我的订单、我的服务、申请服务、我的动态、我的相册、我的预约、VIP会员、设置等

### 3.14 虚拟用户/AI聊天

- **虚拟用户列表**: 预设的AI虚拟陪玩师
- **AI对话**: 与虚拟用户进行AI对话
- **标签管理**: 为虚拟用户分配标签
