## 1. 项目概述

### 1.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | Vue 3 + Vite + `<script setup>` SPA | 纯前端单页应用，全部 UI 集中在 `App.vue` 单文件中 |
| **后端** | Node.js + Express | RESTful API + Socket.IO 实时通信 |
| **数据库 ORM** | Sequelize | MySQL ORM |
| **实时通信** | Socket.IO | 聊天、语聊房、TRTC 信令 |
| **音视频** | TRTC (腾讯实时音视频) | 1v1 音视频通话 |
| **支付** | 微信支付 | 充值、下单 |
| **文件存储** | 本地 multer | 图片/音频/视频上传 |
| **缓存** | Redis | SMS 限流等 |

### 1.2 项目目录结构


## 2. 前端功能拆解

前端采用 **Vue 3 + Vite SPA** 架构，所有 UI 代码集中写在 **`frontend/src/App.vue`** 单文件中（约 6000+ 行）。页面之间通过底部 TabBar 和模态弹窗 (Modal) 进行切换，使用本地状态管理（`reactive()`），暂时未接入后端 API 接口（使用硬编码 Mock 数据）。

### 2.1 整体架构

- **入口文件**: `frontend/src/App.vue`
- **路由方式**: 条件渲染 (v-if/v-show) + `currentTab` 状态变量
- **状态管理**: Vue 3 `reactive()` 本地状态，无 Vuex/Pinia
- **弹窗体系**: 全屏覆盖式 `modal-overlay`，通过 `showXxxModal` 布尔变量控制（共 28 个弹窗）

#### 底部 TabBar 导航 (currentTab)

| Tab | 变量值 | 图标 | 功能 |
|-----|--------|------|------|
| 首页 | `'home'` | 🏠 | 平台首页 |
| 广场 | `'circle'` | 📢 | 动态/圈子 |
| 消息 | `'message'` | 💬 | 消息列表 + 聊天 |
| 我的 | `'mine'` | 👤 | 个人中心 |
