# 测试覆盖率运行指南

## 🚀 快速开始

### Windows 用户

**方式一：使用脚本（推荐）**
```bash

```

**方式二：手动命令**
```bash

```

### macOS / Linux 用户

**方式一：使用脚本（推荐）**
```bash

```

**方式二：手动命令**
```bash
cd frontend
npm install
npm run test:coverage
```

## 🔧 常见问题

### Q1: npm install 失败

**问题**: 网络问题导致依赖下载失败

**解决方案**:
```bash
# 使用淘宝镜像
npm install --registry=https://registry.npmmirror.com

# 或清理缓存后重试
npm cache clean --force
npm install
```

### Q2: 测试报错 "Cannot find module"

**问题**: 依赖未正确安装

**解决方案**:
```bash
# 删除node_modules重新安装
rm -rf node_modules package-lock.json
npm install
```

### Q3: 覆盖率不达标

**问题**: 实际覆盖率低于75%

**解决方案**:
1. 查看coverage/index.html中的未覆盖代码
2. 添加缺失的测试用例
3. 运行 `npm run test:coverage` 重新测试

### Q4: 测试超时

**问题**: 测试运行时间过长

**解决方案**:
```bash
# 增加超时时间 (vitest.config.ts)
test: {
  testTimeout: 20000
}
```

### Q5: 找不到vitest命令

**问题**: vitest未安装

**解决方案**:
```bash
npm install vitest @vue/test-utils jsdom --save-dev
```
