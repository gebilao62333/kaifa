## 📊 三、技术架构

### 核心技术栈
- **框架**: Vue 3
- **状态管理**: Pinia
- **构建工具**: Vite 5
- **样式**: CSS
- **语音SDK**: 腾讯云TRTC
- **测试框架**: Vitest

### 项目结构
```
frontend/
├── src/
│   ├── pages/              # 主包页面（4个）
│   ├── gamePages/          # 游戏陪玩模块（6个页面）
│   ├── circlePages/        # 圈子模块（7个页面）
│   ├── minePages/          # 个人中心模块（4个页面）
│   ├── homePages/          # 首页扩展（1个页面）
│   ├── chatPages/          # 聊天模块（1个页面）
│   ├── components/         # 公共组件（6个）
│   ├── store/              # 状态管理（4个）
│   ├── common/             # 公共模块（4个）
│   ├── tools/              # 工具模块（2个）
│   ├── static/             # 静态资源
│   ├── App.vue             # 应用入口（单页应用）
│   ├── main.js             # 主入口文件
│   ├── pages.json          # 页面路由配置
│   └── manifest.json       # 应用配置
├── tests/                  # 测试文件
├── package.json            # 项目配置
├── vite.config.ts         # Vite配置
└── tsconfig.json           # TypeScript配置
```
